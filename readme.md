# 项目看板

一款基于 Electron 的轻量级项目管理桌面应用，支持看板视图、任务管理、人员管理和环境配置管理，数据本地存储，开箱即用。

## 功能介绍

### 项目管理
- 创建、编辑、删除项目及子项目
- 项目列表与项目详情视图切换
- 支持项目分类与多级子项目结构

### 看板视图
- 任务按状态分列展示（待处理、进行中、已完成等）
- 支持自定义任务类型
- 任务卡片直接删除操作

### 任务管理
- 创建、编辑、删除任务
- 任务指派负责人
- 任务图片上传（支持多图）与分页显示
- 重要任务标记（星标置顶）
- 任务重复机制，支持 5 种频率：
  - 每 N 天重复
  - 每周指定日重复
  - 每月指定日期重复
  - 每周 N 次
  - 每月 N 次

### 重要任务视图
- 独立的重要任务侧边栏入口
- 支持按关键词搜索、任务类型、项目、负责人多维筛选

### 环境配置管理
- 项目级环境配置信息管理（MySQL、Redis、RocketMQ 等）
- 支持开发/测试/生产多环境
- 新建项目自动生成默认配置模板
- 自定义字段扩展
- 配置详情查看与一键复制
- 子项目独立环境配置（文档地址、测试/生产环境地址）

### 人员管理
- 添加、编辑、删除团队成员
- 任务指派时下拉选择负责人

### 其他
- 亮色/暗色主题切换
- 响应式布局，支持自适应窗口大小

## 数据存储

应用采用本地文件系统存储，无需数据库。

### 存储位置

| 运行模式 | 数据目录 |
|---------|---------|
| 开发模式（`npm start`） | 项目根目录下的 `data/` |
| 打包后（exe） | exe 所在目录下的 `data/` |

### 存储结构

```
data/
├── projects/
│   ├── project_<id>.json    # 每个项目一个独立 JSON 文件
│   ├── project_<id>.json
│   └── ...
├── people.json              # 人员信息
├── <taskId>_<index>.jpg     # 任务附件图片
└── ...
```

- **项目数据**：每个项目保存为独立的 JSON 文件，包含项目信息、子项目、任务列表和环境配置
- **人员数据**：统一存储在 `people.json`
- **任务图片**：以 `任务ID_序号.jpg` 格式存储在 data 根目录

## 打包与启动

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装依赖

```bash
npm install
```

### 开发模式启动

```bash
npm start
```

将以 Electron 窗口启动应用，自动打开 DevTools 便于调试。

### 打包为 Windows 桌面程序

```bash
npm run build
```

打包完成后在 `dist/` 目录下生成：

| 文件 | 说明 |
|-----|------|
| `项目看板 Setup 1.0.0.exe` | NSIS 安装程序，支持自定义安装路径、创建桌面快捷方式 |
| `项目看板 1.0.0.exe` | 便携版，免安装直接运行 |
| `win-unpacked/` | 解压目录，包含完整运行文件 |

如果只需要生成解压目录（不打包为安装程序），可以运行：

```bash
npm run build:dir
```

### 项目结构

```
├── main.js          # Electron 主进程，窗口创建与 IPC 文件操作
├── preload.js       # 预加载脚本，安全暴露 API 到渲染进程
├── kanban.html      # 应用主页面（SPA，含 HTML/CSS/JS）
├── package.json     # 项目配置与打包配置
└── data/            # 本地数据存储目录
```
